# 📎 RE:코더



## 👀 서비스 소개
* 서비스명:   Wellness Trip Planner
* 서비스설명:  건강하고(wellbeing) + 행복한(happiness) 전라남도 여행을 위한 스케줄러 프로그램
<br>

## 📅 프로젝트 기간
2024.02.23 ~ 2022.04.04 (6주)
<br>

## ⭐ 주요 기능
* 기능1 : 회원가입 및 로그인
* 기능2 : 여행 계획
* 기능3 : 게시판
* 기능4 : 준비물
* 기능5 : 여행계획 전체조회
<br>
